var – The old way to declare variables. You can change its value and even declare it again.

let – The modern way to declare variables. You can change its value, but you cannot declare it twice in the same scope.

const – Declares a variable whose value cannot be changed after assignment.

map() – Loops through an array and returns a new array with the modified values.

filter() – Creates a new array containing only the items that match a certain condition.

forEach() – Loops through an array to perform an action on each item, but does not return a new array.

Arrow functions (=>) – A shorter, cleaner way to write functions.

Template literals (`) – Allows embedding variables or expressions inside strings easily using backticks.
